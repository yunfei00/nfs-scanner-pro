# 高保真静态原型 — Release 010

## 打开方式

在仓库根目录，用浏览器直接打开：

```text
prototypes/high_fidelity/index.html
```

或双击 `index.html`（Windows 资源管理器）。

## 要求

- 无需网络、无 CDN、无外链图片
- 页面按 **1920×1080** 设计；窗口较小时可滚动查看
- 左侧导航 **Hover** 展开 56px → 180px

## 文件

| 文件 | 说明 |
|---|---|
| `index.html` | 主窗口高保真 |
| `styles.css` | Token CSS 变量与布局 |
