# 02 Product Goals

## Primary Goal

Build a commercial near-field scanning product for PCB test work.

The product should guide the user through this flow:

```text
Project -> Devices -> Sample -> Region -> Scan -> Analysis -> Report
```

## Goals

- Make the workflow clear for test engineers.
- Keep project data organized.
- Make scan results reproducible.
- Support reusable device profiles.
- Treat camera as optional.
- Support Hx and Hy probe channels.
- Support multiple Regions in one Project.

## Non Goals

The product is not just a motion controller, spectrum controller, camera viewer, or heatmap viewer.

It is a complete near-field scanning workflow platform.
