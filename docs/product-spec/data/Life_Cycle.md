# Life Cycle

## Project Life Cycle

- Created：项目已创建。
- Configured：样品和基础信息已配置。
- Active：项目正在使用。
- Completed：主要测试完成。
- Archived：项目归档。

## Region Life Cycle

- Created：Region 已创建并命名。
- Positioned：起点和终点已记录。
- Aligned：区域对齐已完成，可选。
- Configured：扫描参数已配置。
- Scanned：已有扫描数据。
- Analyzed：已有分析结果。
- Reported：已进入报告。

## Scan Life Cycle

- Pending：等待执行。
- Running：正在执行。
- Paused：暂停。
- Completed：完成。
- Failed：失败。
- Cancelled：取消。
- Verified：结果已确认。

## Alignment Life Cycle

- Empty：未设置。
- Draft：编辑中。
- Valid：已确认。
- Updated：已修改。
- Disabled：不使用对齐。

## Analysis Life Cycle

- Created：分析记录已创建。
- Configured：分析参数已设置。
- Generated：分析结果已生成。
- Reviewed：用户已查看确认。
- Exported：已导出。

## Report Life Cycle

- Draft：报告草稿。
- Generated：报告已生成。
- Reviewed：报告已检查。
- Exported：报告已导出。
- Archived：报告归档。
