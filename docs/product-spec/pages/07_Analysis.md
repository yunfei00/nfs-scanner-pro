# 07 Analysis

## Page Purpose

Analysis page is used after scan data exists.

It helps the user inspect heatmaps, spectrum data, and results.

## User Goals

- Open scan result.
- Select Region.
- Select Hx or Hy.
- View heatmap.
- Change frequency point.
- Change trace.
- Adjust color map and range.
- Compare Hx and Hy.
- Export analysis images.

## Analysis Inputs

- Project
- Region
- Scan data
- Alignment data
- Frequency data
- Trace data

## Heatmap Modes

Supported result modes:

- Hx heatmap
- Hy heatmap
- Total heatmap
- Magnitude
- Phase
- Real part
- Imaginary part

## Camera Overlay

If alignment exists, heatmap can be overlaid on the sample image.

If alignment does not exist, heatmap can still be shown as data grid result.

## Rules

- Analysis does not control devices.
- Analysis does not change raw scan data.
- Analysis settings can be saved as analysis result.
