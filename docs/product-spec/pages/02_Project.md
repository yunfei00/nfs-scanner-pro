# 02 Project

## Page Purpose

Project page manages Project creation and Project information.

A Project represents one PCB or one tested object.

## User Goals

- Create a new Project.
- Open an existing Project.
- View Project information.
- See Regions under the current Project.

## Project Information

- Project name
- Project ID
- Customer
- Sample name
- Sample ID
- Operator
- Created time
- Updated time
- Notes

## Page Actions

- New Project
- Open Project
- Duplicate Project
- Archive Project

## Rules

- Project page does not control devices.
- Project page does not start scans.
- A Project can contain multiple Regions.
