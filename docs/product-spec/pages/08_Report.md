# 08 Report

## Page Purpose

Report page creates professional output from Project, Region, Scan, and Analysis data.

## User Goals

- Select report template.
- Select Regions to include.
- Select heatmaps and figures.
- Write conclusion and notes.
- Export report.

## Report Content

A report should include:

- Project information
- Sample information
- Device snapshot
- Region information
- Scan parameters
- Heatmap images
- Analysis result
- Conclusion

## Export Types

First version should support:

- PDF
- Word
- Excel
- Images

## Rules

- Report is generated from saved Project data.
- Report should not modify raw scan data.
- Export history should be saved.
