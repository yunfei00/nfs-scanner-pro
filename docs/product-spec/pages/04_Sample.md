# 04 Sample

## Page Purpose

Sample page manages the PCB or tested object information.

## User Goals

- View current sample information.
- Edit sample name and sample ID.
- Add sample notes.
- Capture or import a sample image.
- Confirm that the PCB is placed correctly.

## Sample Information

- Sample name
- Sample ID
- Hardware version
- Software version
- Description
- Image

## Camera Relationship

If camera is available, the user can capture a sample image.

If camera is not available, the user can continue without image overlay.

## Rules

- Sample belongs to Project.
- A Project has one main Sample.
- Sample image belongs to Project.
- Region alignment uses the Sample image.
