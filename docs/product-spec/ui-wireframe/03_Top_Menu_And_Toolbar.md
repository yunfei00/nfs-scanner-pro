## 工具栏默认按钮

| 按钮 | 是否默认显示 | 原因 |
|---|---|---|
| 开始扫描 | 是 | 高频核心动作 |
| 停止扫描 | 是 | 安全动作 |
| 拍照 | 是 | 扫描前常用 |
| 区域对齐 | 是 | 扫描前常用 |
| 网格 | 是 | 画布辅助 |
| 测量 | 是 | 画布辅助 |
| 连接设备 | 否 | 通过设备状态点击或工具菜单进入 |
| 新建/打开/保存项目 | 否 | 放入文件菜单 |
