# 05 扫描页面线框

扫描页面是系统默认首页和核心页面。工程师 90% 时间停留在此页面。

## 画布内容

- PCB 照片
- 热力图叠加
- 扫描区域矩形
- 扫描网格点
- 坐标轴
- 小地图
- 鼠标当前位置浮窗
- Breadcrumb 当前上下文

## Breadcrumb

```text
项目 > CPU_Area > Hx Probe > 2.450 GHz > 6461 pts
```

## 默认隐藏

日志、频谱曲线、扫描统计、3D 曲面、数据表格全部默认隐藏，通过“视图”菜单打开。

## 右侧参数 Dock

宽度 340 px，用户可关闭或自动隐藏。

```text
扫描设置     展开
区域设置     展开
显示设置     折叠
仪表设置     折叠
高级设置     折叠
```
