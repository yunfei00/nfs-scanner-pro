# 01 主窗口线框：1920 × 1080

## 固定尺寸

| 区域 | 尺寸 |
|---|---:|
| 窗口基准 | 1920 × 1080 |
| 顶部菜单栏 | 32 px |
| 顶部工具栏 | 56 px |
| 设备状态栏 | 40 px |
| 左侧导航收起宽度 | 56 px |
| 左侧导航展开宽度 | 180 px |
| 右侧参数 Dock 默认宽度 | 340 px |
| 右侧参数 Dock 收起宽度 | 40 px |
| 底部状态栏 | 32 px |
| 主画布最小占比 | ≥ 70% 宽度 |

## 默认布局

```text
┌────────────────────────────────────────────────────────────────────┐
│ 菜单栏：文件 编辑 视图 工具 设置 帮助                               │ 32
├────────────────────────────────────────────────────────────────────┤
│ 工具栏：开始扫描 停止扫描 拍照 区域对齐 网格 测量                   │ 56
├────────────────────────────────────────────────────────────────────┤
│ 设备状态：●运动平台 ●频谱仪 ●相机 ●舵机 | 探头 高度 区域 频率 点数   │ 40
├──────┬──────────────────────────────────────────────┬──────────────┤
│ 图标 │                                              │ 参数 Dock     │
│ 导航 │                 PCB 工作画布                  │ 340 / AutoHide│
│ 56px │                                              │              │
├──────┴──────────────────────────────────────────────┴──────────────┤
│ 状态栏：状态 扫描进度 扫描点 已用时间 剩余时间 日期 时间             │ 32
└────────────────────────────────────────────────────────────────────┘
```

## 顶部菜单

```text
文件(F)  编辑(E)  视图(V)  工具(T)  设置(S)  帮助(H)
```

## 工具栏默认按钮

```text
开始扫描 | 停止扫描 | 拍照 | 区域对齐 | 网格 | 测量
```

不在工具栏放：新建项目、打开项目、保存项目、项目管理器。
