# 02 左侧导航线框

## 一级模块

```text
扫描
设备
分析
报告
```

## 默认状态：图标模式

宽度：56 px。

```text
┌──────┐
│  ◎   │ 扫描
│  ▣   │ 设备
│  📈  │ 分析
│  📄  │ 报告
└──────┘
```

实际界面只显示图标，不显示文字。

## 鼠标悬停：展开模式

宽度：180 px，动画 150~200 ms。

```text
┌──────────────────┐
│ ◎  扫描          │
│ ▣  设备          │
│ 📈 分析          │
│ 📄 报告          │
└──────────────────┘
```

规则：鼠标进入展开，离开收起，当前页面高亮，不设置“项目”入口。
