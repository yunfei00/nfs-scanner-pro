# 00 Introduction

## Purpose

This document set defines the product direction for NFS Scanner Professional.

It is written before implementation so that product, design, architecture, and code can follow the same plan.

## Product Type

NFS Scanner Professional is a desktop industrial test application.

It is used for near-field scanning of PCB samples and for generating heatmap based analysis results.

## Target Users

- EMC engineers
- RF engineers
- Hardware engineers
- Test engineers
- Lab operators

## Working Context

The product is designed for lab and engineering use.

Typical work:

1. Put a PCB on the platform.
2. Connect motion and spectrum devices.
3. Select or create a Project.
4. Create scan Regions.
5. Run scans.
6. Analyze heatmaps.
7. Export reports.

## Product Rule

Design first. Code later.
