# 状态说明 — 设备异常 (Device Error)

Release 010.5 · 设备连接或通信失败

## 1. 状态名称

**设备异常** — 任一必需设备（运动平台 / 频谱仪）断开或报错；相机离线 **不** 触发此全局态（ADR-0003）。

## 2. 顶部状态显示

| 位置 | 文案 | 颜色 |
|---|---|---|
| 菜单栏右侧 | 状态：设备异常 | 红色 `#EF4444` |
| 设备状态栏 | **对应设备**状态灯变红 | `#EF4444` |
| 其余设备 | 仍可为绿点 | — |

示例：运动平台断开 → `运动平台(COM6)` 红点，文案可追加「未连接」。

## 3. 按钮可用性

| 控件 | 状态 |
|---|---|
| 开始扫描 | **禁用** |
| 停止扫描 | 视当前是否在扫描中 |
| 设备页 → 重新连接 | **启用** |
| 查看日志 | **启用**（占位，打开日志 Dock） |

## 4. 底部状态栏

```
状态：设备异常 — 运动平台未连接
扫描进度：— 或冻结上次值
扫描点：— 
```

## 5. 右侧 Dock

- 扫描参数 **只读** 或整体禁用
- 设备页 Dock 显示错误详情与重连

## 6. 画布

- PCB 仍可显示上次热力图或灰态占位
- 可选半透明遮罩 + 错误图标

## 7. 日志 / 提示

- **错误 Banner** 或对话框：「运动平台 COM6 连接失败」
- 提供 **重新连接**、**查看日志** 入口
- 日志 Dock 经用户操作打开（仍非默认显示）

## 8. Release 011 实现

- `deviceStatusMotionIndicator` 设 `status=error` + QSS 红色
- 菜单栏 `status-lamp` 红色
- Mock 断连场景切换，不接真实串口
