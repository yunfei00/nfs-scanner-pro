# 状态说明 — 扫描中 (Scanning)

Release 010.5 · 扫描进行中 · 参考扫描页框架

## 1. 状态名称

**扫描中** — ScanTask 处于「扫描中」，运动平台按路径采点，热力图实时更新（Mock 表现为整图刷新）。

## 2. 顶部状态显示

| 位置 | 文案 | 颜色 |
|---|---|---|
| 菜单栏右侧 | 状态：扫描中 | 蓝色脉冲或 `#1A8CFF` |
| 设备状态栏 | 运动平台绿点；频谱仪可显示忙碌 | 绿 / 琥珀 |

## 3. 按钮可用性

| 控件 | 状态 |
|---|---|
| 开始扫描 | **禁用** |
| 停止扫描 | **启用**（红色危险按钮） |
| 拍照 / 对齐 | **禁用** |
| 网格 / 测量 | 可选启用（Mock 可保持禁用） |
| 文件 → 关闭项目 | **禁用** |

## 4. 底部状态栏

```
状态：扫描中
扫描进度：42%（进度条填充，约 10px 高）
扫描点：2714 / 6461
预计剩余时间：00:07:05
```

## 5. 右侧 Dock

- **扫描参数：只读**（input readonly、select disabled）
- 区域设置不可修改
- Dock 仍可滚动查看

## 6. 画布

- PCB 上热力图 **整图 QPixmap 更新**（禁止逐格 Item）
- 扫描路径/当前点可高亮（Mock 可选）
- 坐标浮窗随当前点更新
- 色带、小地图保持可见

## 7. 日志 / 提示

- 可选：底部状态栏消息轮播采点日志
- 日志 Dock 仍默认隐藏；经视图菜单可开

## 8. Release 011 实现

- Mock 定时器更新进度条与点数
- `_status.set_state("扫描中")`；参数 Dock 设 readonly
- 不接真实扫描线程
