# 状态说明 — 准备就绪 (Ready)

Release 010.5 · 默认扫描页状态 · 原型：[index.html](../../../../prototypes/high_fidelity/index.html)

## 1. 状态名称

**准备就绪** — ScanTask 处于「准备就绪」，设备已连接，可开始扫描。

## 2. 顶部状态显示

| 位置 | 文案 | 颜色 |
|---|---|---|
| 菜单栏右侧 | 状态：准备就绪 | 绿色指示灯 `#22C55E` |
| 设备状态栏 | 四设备绿点 | `#22C55E` |

## 3. 按钮可用性

| 控件 | 状态 |
|---|---|
| 开始扫描 | **启用**（主按钮绿色/蓝色） |
| 停止扫描 | **禁用**或灰色 |
| 拍照 / 对齐 / 网格 / 测量 | **启用** |
| 文件菜单项目操作 | **启用** |
