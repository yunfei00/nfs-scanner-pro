# 主窗口高保真规格 — 1920×1080（v2 工业仪器级）

> Release 010 · 布局在 Release 008 线框基础上 **视觉细化**，尺寸以本规格为准  
> 线框权威：[ui-wireframe/01_Main_Window_1920x1080.md](../../ui-wireframe/01_Main_Window_1920x1080.md)

## 1. 主窗口尺寸与视口

| 项 | 值 |
|---|---|
| 设计基准 | **1920 × 1080** |
| 浏览器适配 | `transform: scale()` 撑满视口，比例不变 |
| 原型入口 | [prototypes/high_fidelity/index.html](../../../../prototypes/high_fidelity/index.html) |
| SVG 稿 | [assets/main_window_high_fidelity.svg](../assets/main_window_high_fidelity.svg) |

## 2. 固定区域尺寸（v2）

| 区域 | 尺寸 |
|---|---:|
| 顶部菜单栏 | **36 px** |
| 顶部工具栏 | **64 px** |
| 设备状态栏 | **48 px** |
| 左侧导航收起 | **64 px** |
| 左侧导航展开 | **180 px** |
| 右侧参数 Dock | **360 px** |
| 底部状态栏 | **40 px** |

主画布宽度：1920 − 64 − 360 = **1496 px**（≥ **75%** 主内容宽）

## 3. 顶部菜单栏

- 字号 **14~15px**，清晰可读
- 菜单：文件(F) · 编辑(E) · 视图(V) · 工具(T) · 设置(S) · 帮助(H)
- **右侧**：绿色状态灯 + **状态：准备就绪**（禁止英文 READY）
- 项目 CRUD **仅**在文件菜单（ADR-0013）

## 4. 顶部工具栏

工业仪器质感：渐变按钮、图标 + 文字、高度 **38~42px**。

| 按钮 | 样式 |
|---|---|
| 开始扫描 | 绿色/蓝色主按钮 + 播放图标 |
| 停止扫描 | 红色危险按钮 |
| 拍照 / 区域对齐 / 网格 / 测量 | 深色描边按钮 |

**禁止**：新建项目、打开项目、保存项目、项目管理器。

## 5. 左侧导航

- 仅：**扫描 · 设备 · 分析 · 报告**
- 默认 **64px** 图标；Hover **180px**（~180ms）
- 图标 **24~26px**；选中项蓝色背景 + 左侧高亮线
- **禁止**：项目、设置、帮助

## 6. 设备状态栏（单行）

芯片式设备标签 + 右侧元数据，**不换行**：

```text
● 运动平台(COM6)  ● 频谱仪(ZNA67)  ● 相机(USB3.0)  ● 舵机系统
探头：Hx(100 μm)  高度：5.000 mm  区域：CPU_Area  频率：2.450 GHz  点数：6461
```

## 7. 中央 PCB 画布（视觉核心）

**PCB 必须是最大视觉中心**，占主区域绝大部分。

### 7.1 PCB 质感（SVG）

| 元素 | 要求 |
|---|---|
| 基板 | 深绿色渐变、圆角外形 |
| 芯片 | 多颗 IC，带引脚示意 |
| 焊盘/过孔 | 金色，**数量多**（脚本/图案生成） |
| 走线 | 金色主走线 + 浅色辅助走线 |
| 安装孔 | 四角金属化孔 |
| 接插件 | 左右边缘连接器区域 |
| 丝印 |  subtle 点阵 |

### 7.2 扫描与热力图

| 元素 | 要求 |
|---|---|
| 扫描区域 | **白色**矩形框 + 四角/边中 **拖拽手柄** |
| 热力图 | **radial-gradient** 整层叠加：中心红 → 黄 → 绿 → 蓝 |
| 扫描网格 | pattern 点阵，非热力数据格 |
| 禁止 | 逐格 rect 模拟热力图 |

### 7.3 叠加 UI

| 元素 | 位置 |
|---|---|
| 面包屑 | 画布左上浮层，14px |
| 色带 | 画布右侧，标题 **幅度(dBm)**，刻度 -10…-90 |
| 当前坐标 | 浮窗 13~14px，含 X/Y/Z/频率/幅度 |
| 小地图 | 左下角 |

面包屑：`项目 > CPU_Area > Hx Probe > 2.450 GHz > 6461 pts`

## 8. 右侧扫描参数 Dock（360px）

标题：**扫描参数**

| 分组 | 默认 | 内容要点 |
|---|---|---|
| 扫描设置 | 展开 | 下拉框、输入框、checkbox 统一样式 |
| 区域设置 | 展开 | 起点/终点/步长；91×71=6461 等统计卡片 |
| 显示/仪表/高级 | 折叠 | 占位 |

区域设置示例值：

- 起点：10.00 / -150.00 / 5.00  
- 终点：190.00 / -10.00 / 5.00  
- 步长：2.00 / 2.00 / 0.00  

## 9. 底部状态栏（40px）

含 **进度条** + 清晰 13px 文案：

```text
状态：准备就绪 | 扫描进度：0% [====    ] | 扫描点：0 / 6461 | 预计剩余时间：00:12:31 | 日期 | 时间
```

## 10. 默认隐藏

日志、频谱、统计 Dock — 主屏 **不渲染**。

## 11. 视觉质量门禁

高保真 **不得** 像线框 demo：需有渐变、阴影、工业控件质感、真实 PCB 细节与清晰热力图。
