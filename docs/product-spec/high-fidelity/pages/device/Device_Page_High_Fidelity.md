# 设备页高保真 — Device Page

Release 010.5 · 原型：[device.html](../../../../prototypes/high_fidelity/device.html)

## 1. 页面定位

设备一级导航页：管理 **DeviceProfile** 下四设备连接与基础控制，**不接真实硬件**。

## 2. 框架一致性

与扫描页相同：

- 菜单栏 / 工具栏 / 设备状态栏 / 左侧导航 / 底部状态栏
- 1920×1080 · `styles.css` · 无外部依赖
- 左侧导航选中 **设备**；**无「项目」**导航项

## 3. 中央内容 — 设备中心

2×2 设备卡片网格：

| 卡片 | 必含元素 |
|---|---|
| **运动平台** | COM6、115200、已连接、X/Y/Z 坐标、回零、Jog |
| **频谱仪** | ZNA67、TCP/IP、地址、Trace、频率范围、测试连接 |
| **相机** | USB3.0、分辨率、拍照、预览区 |
| **舵机系统** | Hx/Hy 状态、旋转角度、偏移补偿、Hx/Hy 校准入口 |

## 4. 右侧 Dock — 设备配置

- 标题：**设备配置**（非「扫描参数」）
- DeviceProfile 选择、连接策略、高级折叠占位
