# 报告页高保真 — Report Page

Release 010.5 · 原型：[report.html](../../../../prototypes/high_fidelity/report.html)

## 1. 页面定位

报告一级导航：报告列表、预览、导出设置。**不生成真实 PDF**。

## 2. 框架一致性

- 相同菜单栏、设备状态栏、左侧导航、底部状态栏
- 左侧导航选中 **报告**
- 工具栏为报告动作：新建 / 预览 / 导出 PDF·Word·Excel

## 3. 中央内容 — 报告工作区

| 区域 | 内容 |
|---|---|
| **报告列表**（左） | 报告名称、日期、探头 |
| **报告预览**（中） | 报告名称、项目、区域、扫描时间、Hx/Hy、热力图缩略图、结论摘要 |

## 4. 右侧 Dock — 报告设置

- 模板、Logo、导出质量
- 包含热力图 / 设备信息勾选
- 静态说明：不生成真实 PDF
