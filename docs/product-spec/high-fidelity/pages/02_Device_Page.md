# 设备页 — 设计说明（占位）

## 1. 页面目标

展示与管理运动平台、频谱仪、相机、舵机连接状态与 DeviceProfile。

## 2. 使用场景

切换左侧 **设备** 导航；连接/断开设备、查看端口与型号。

## 3. 页面结构（规划）

设备卡片列表 + 连接按钮 + 状态指示（单行状态栏与之联动）。

## 4. 显示内容

四设备状态、端口、型号、Mock 连接按钮。

## 5. 默认隐藏

日志 Dock、频谱实时面板（扫描页或视图菜单打开）。

## 6. 与 Dock 关系

参数 Dock 在设备页可收起或显示设备相关属性（Release 011+）。

## 7. 与菜单栏

**设备** 相关批量操作可放 **工具** 或 **设置** 菜单。

## 8. 领域对象

DeviceProfile、DeviceSnapshot、Device 状态机。

## 9. PySide6 建议

`devicePage` + Mock `Connect_Devices`；Camera 可选不阻断扫描（ADR-0003）。

## 10. 高保真

本 Release **未**单独出设备页 HTML；导航项已在主原型中预留。
