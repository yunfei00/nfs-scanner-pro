# 报告页 — 设计说明（占位）

## 1. 页面目标

预览与导出 EMC 扫描报告（Report 对象）。

## 2. 使用场景

分析完成后生成 PDF/HTML 报告。

## 3. 页面结构（规划）

报告列表 + 预览区 + 导出按钮。

## 4. 显示内容

Report 标题、日期、关联 Region/ScanTask Mock。

## 5. 默认隐藏

数据表格 Dock。

## 6. 与 Dock

报告属性可复用右侧 Dock 或独立对话框。

## 7. 与菜单栏

**文件** 菜单导出；**报告** 菜单模板设置。

## 8. 领域对象

Report、Report 状态机、Region/ScanTask 引用。

## 9. PySide6 建议

`reportPage` + Mock 导出路径选择。

## 10. 高保真

本 Release 仅占位说明。
