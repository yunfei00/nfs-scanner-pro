## 字号

| 用途 | 字号 |
|---|---:|
| 顶部菜单 | 14 |
| 工具栏按钮 | 14 |
| 面板标题 | 16 |
| 表单标签 | 13 |
| 表单内容 | 13 |
| 状态栏 | 13 |
| 画布浮窗 | 13 |

间距采用 8px 栅格：4 / 8 / 12 / 16 / 24 / 32。
