# Release_008_UI_Wireframe

本包用于覆盖到 `nfs-scanner-pro` 仓库，目标是停止反复生成高保真，先固定主窗口线框、Dock 布局、尺寸规范、Qt/PySide6 实现规范。

## 手机 Termux 覆盖方式

```bash
cd ~/nfs-scanner-pro
unzip -o /sdcard/Download/Release_008_UI_Wireframe.zip -d .
git add docs scripts README.md
git commit -m "docs: add ui wireframe and design system release 008"
git push
```
